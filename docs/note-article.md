## Docs Summary to Image とは？

**ノートの内容をAIが理解し、各セクションごとに美しいサマリー画像を自動生成**するObsidianプラグインです。

### 主な特徴

**1. ワンクリックで画像生成**
コマンドパレットから「Generate Summary Images」を選ぶだけ。あとはAIにお任せ。

**2. 見出しベースの自動分割**
ノート内の見出し（H2, H3）を検出し、セクションごとに最適な画像を生成。最大8枚まで対応。

**3. 最新AIモデルを採用**
- **OpenAI GPT-5-mini**: ノート内容を深く理解し、的確な画像プロンプトを生成
- **kie.ai nano-banana-pro**: 高品質なビジュアルを高速生成

**4. Liquid Glass スタイル**
トレンドの「Liquid Glass」デザインを採用。透明感のある美しいビジュアルが、あなたのノートを格上げします。

## 1. インストール方法

### 1.1 プラグインファイルの配置

1. Obsidianの設定 → コミュニティプラグイン → 「制限モードをオフにする」
2. Vaultフォルダ内の `.obsidian/plugins/` に `docs-summary-to-image` フォルダを作成
3. 以下のファイルを配置：
   - `main.js`
   - `manifest.json`
   - `styles.css`

4. Obsidianを再起動
5. 設定 → コミュニティプラグイン → 「Docs Summary to Image」を有効化

### 1.2 必要なAPIキーの取得

#### OpenAI API Key
1. [OpenAI Platform](https://platform.openai.com/) にアクセス
2. アカウント作成またはログイン
3. API Keys → Create new secret key
4. 生成されたキー（`sk-...`）をコピー

#### kie.ai API Key
1. [kie.ai](https://kie.ai/) にアクセス
2. アカウント作成
3. API設定からAPIキーを取得

---

## 2. 設定方法

### 2.1 プラグイン設定を開く

Obsidian設定 → コミュニティプラグイン → Docs Summary to Image → 歯車アイコン

### 2.2 API設定

| 設定項目 | 説明 |
|---------|------|
| **API Mode** | `Direct API` を選択 |
| **OpenAI API Key** | 取得したOpenAIのAPIキーを入力 |
| **kie.ai API Key** | 取得したkie.aiのAPIキーを入力 |

**セキュリティ機能**: APIキーは`••••••`でマスク表示されます。👁️アイコンで表示/非表示を切り替え可能。

### 2.3 画像設定

| 設定項目 | 説明 | おすすめ |
|---------|------|---------|
| **Aspect Ratio** | 画像の縦横比 | `16:9`（ブログ向け）または `1:1`（SNS向け） |
| **Max Images** | 最大生成枚数 | `8` |
| **Image Folder** | 保存先フォルダ | `assets/ai-images` |

## 3. 使い方

### 3.1 基本的な使い方

1. 画像を生成したいノートを開く
2. `Cmd/Ctrl + P` でコマンドパレットを開く
3. 「Generate Summary Images」を選択
4. 進捗バーが表示され、画像が生成される
5. 各見出しの下に画像が自動挿入される

### 3.2 効果的な使い方のコツ

**見出しを明確に**
```markdown
## 悪い例
内容がここに...

## 良い例：AIで作業効率を10倍にする方法
内容がここに...
```

見出しが具体的であるほど、AIは的確な画像を生成します。

**セクションの内容を充実させる**
各セクションに100文字以上の内容があると、より精度の高い画像が生成されます。

## 4. トラブルシューティング

### Q: 画像が生成されない

**A:** 以下を確認してください：
- APIキーが正しく設定されているか
- インターネット接続があるか
- ノートに見出し（##）があるか

### Q: エラーが表示される

**A:** コンソールログ（`Cmd/Ctrl + Shift + I`）を確認し、エラーメッセージを確認してください。

よくあるエラー：
- `OpenAI API error: 401` → APIキーが無効
- `kie.ai API error: 429` → レート制限。しばらく待ってから再試行

### Q: 画像の品質を上げたい

**A:** kie.aiの設定で解像度を変更できます（デフォルト: 1K）。
